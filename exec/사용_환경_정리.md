## AI

- **Python:** 3.12.3
- **패키지:** `ai/requirements.txt`
- **실행 엔트리:** `ai/src/main.py`

* **기본 포트:** 8000

> 설치/실행

- 로컬 실행:

```bash
cd ai
pip install -r requirements.txt
uvicorn src.main:app --reload --port 8000
```

- EC2:

```bash
sudo systemctl start closetory-ai
# 변경 반영/재배포 시
sudo systemctl restart closetory-ai
```

> 확인 명령어

```bash
python3 --version
sudo systemctl status closetory-ai --no-pager
curl -i http://127.0.0.1:8000/
```

## 환경 변수

### Database Configuration

- DB_URL=JDBC URL
- DB_USERNAME=DB 유저 이름
- DB_PASSWORD=DB 유저 비밀번호

### Redis Configuration

- REDIS_HOST=Redis 호스트
- REDIS_PORT=Redis 포트
- REDIS_PASSWORD=Redis 비밀번호

### S3

- AWS_ACCESS_KEY_ID=AWS S3 접근을 위한 액세스 키 ID
- AWS_SECRET_ACCESS_KEY=AWS S3 접근을 위한 시크릿 액세스 키

### Front

- CLOSETORY_BASE_URL=앱에서 API 요청을 보낼 서버 Base URL
- X_ACCESS_TOKEN=액세스 토큰을 담는 HTTP 헤더 이름
- X_REFRESH_TOKEN=리프레시 토큰을 담는 HTTP 헤더 이름
- USERID=사용자 ID를 전달/저장할 때 사용하는 키 이름
- SHARED_PREFERENCES_NAME=앱에서 사용하는 SharedPreferences 파일 이름
- COOKIES_KEY_NAME=쿠키를 SharedPreferences에 저장할 때 쓰는 key 이름

### AI

- API_KEY=제미나이 API 키
