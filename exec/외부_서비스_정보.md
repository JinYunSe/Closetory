# Closetory 외부 서비스 정보

## 개요

본 문서는 Closetory 프로젝트에서 사용하는 외부 서비스 정보를 정리합니다.
AI 생성, 이미지 저장, 캐시/토큰 관리 등을 위해 필요한 정보를 포함합니다.

## 1. AI 서비스

### 1.1 FastAPI AI Server

- **서비스 용도**: AI 코디 추천, 가상 피팅 이미지 생성, 옷 사진 배경 제거 및 보정
- **연동 방식**: Spring Boot → FastAPI HTTP 호출
- **서비스 클래스**: `LookServiceImpl.java`, `ClothesServiceImpl.java`

### 1.2 Gemini API (FastAPI 내부에서 사용)

- **서비스 용도**: 이미지 생성/편집
- **사용 모델**:
  - `gemini-3-pro-image-preview` (Nano Banana Pro)
- **비고**: Gemini API 호출은 FastAPI 서버 내부에서 수행

## 2. 데이터베이스 서비스

### 2.1 MySQL

- **서비스 용도**: 메인 데이터베이스
- **연결 정보**:
  - 데이터베이스: `closetory`

### 2.2 Redis

- **서비스 용도**: JWT 토큰 관리, 통계 캐싱

## 3. 객체 스토리지

### 3.1 S3

- **서비스 용도**: 옷, 룩, 게시글 이미지 저장을 위한 스토리지
- **버킷/리전**:
  - Bucket: `closetory-s3`
  - Region: `ap-northeast-2`
